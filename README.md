## Overview
Implementation of various applications
- Run-length encoding
- Infix / Postfix converter
- Check for balanced parentheses
- Triangular number
- Factorial
- Anagram
- Tower of Hanoi
- Letter frequency
- Base64 encode
- Word count
- Random name generator
- Palindrome checker
- Leap year
- Pig Latin
- Temperature converter
- Times table
- Prime number
- Catalan number
- Reverse integer
- Roman to decimal converter
